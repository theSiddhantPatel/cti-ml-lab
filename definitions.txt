Flower is a Python-based framework that makes it easy to build and run federated learning systems.

Federated Learning is a machine learning approach where multiple clients (devices, servers, or organizations) collaboratively train a model without sharing raw data — only model updates are exchanged.

Flower handles the communication, orchestration, and aggregation between clients and the central server.
